## 🔥 NÄCHSTER SCHRITT (JETZT AUSFÜHREN)

### **SCHRITT 1 — Supabase CLI installieren**

Falls noch nicht geschehen:

```bash
npm install -g supabase
```

Danach prüfen:

```bash
supabase --version
```

---

### **SCHRITT 2 — Projekt verknüpfen**

Im Root deines Projekts:

```bash
supabase link --project-ref DEIN_PROJECT_REF
```

📌 *Den Project-Ref findest du im Supabase Dashboard → Project Settings.*

---

### **SCHRITT 3 — OpenAI API Key setzen (KRITISCH)**

```bash
supabase secrets set OPENAI_API_KEY=sk-xxxxxxxx
```

❗ Wichtig:

* kein `.env`
* kein Client-Code
* **nur** Supabase Secrets

---

### **SCHRITT 4 — Edge Function deployen (DER MOMENT)**

```bash
supabase functions deploy enrich-explanation
```

**Erwartete Ausgabe:**

```
Deploying function enrich-explanation...
Deployed successfully
```

👉 Wenn hier **irgendein Fehler** kommt:
➡️ **sofort stoppen und mir exakt die Fehlermeldung schicken.**
