### **SCHRITT 2 — Supabase CLI installieren (falls noch nicht)**

```bash
npm install -g supabase

```

---

### **SCHRITT 3 — Projekt mit Supabase verknüpfen**

```bash
supabase link --project-ref DEIN_PROJECT_REF
```

📍 *Project Ref findest du im Supabase Dashboard → Project Settings.*

---

### **SCHRITT 4 — OpenAI API Key setzen**

```bash
supabase secrets set OPENAI_API_KEY=sk-xxxx
```

⚠️ **Ganz wichtig**

* kein `.env`
* kein Client-Code
* **nur** über Supabase Secrets
