## ✅ PHASE 1 — Deployment (wir starten JETZT)

Bitte arbeite diese Schritte **genau in dieser Reihenfolge** ab.

### **1️⃣ Supabase CLI installieren (falls noch nicht vorhanden)**

```bash
npm install -g supabase
```

👉 Danach kurz prüfen:

```bash
supabase --version
```

---

### **2️⃣ Projekt mit Supabase verknüpfen**

Im Root deines Projekts:

```bash
supabase link --project-ref DEIN_PROJECT_REF
```

💡 *DEIN_PROJECT_REF findest du im Supabase Dashboard → Project Settings.*

---

### **3️⃣ OpenAI API Key als Secret setzen**

```bash
supabase secrets set OPENAI_API_KEY=sk-xxxx
```

👉 **Wichtig:**

* Kein `.env`
* Kein Client-Code
* **Nur** über `supabase secrets`

---

### **4️⃣ Edge Function deployen**

```bash
supabase functions deploy enrich-explanation
```

**Erwartete Ausgabe:**

```
Deploying function enrich-explanation...
Deployed successfully
```

👉 **Wenn hier ein Fehler kommt: sofort stoppen und melden.**

---

## ✅ PHASE 2 — Verifikation (entscheidend!)

### **5️⃣ Supabase Dashboard → Functions**

* Öffne **Functions**
* Klick auf **enrich-explanation**
* Öffne **Logs**

⚠️ Noch siehst du evtl. nichts – das ist normal.

### **6️⃣ App neu starten (wichtig!)**
