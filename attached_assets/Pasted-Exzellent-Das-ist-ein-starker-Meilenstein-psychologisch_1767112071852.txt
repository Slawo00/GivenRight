Amazon is NOT a recommendation engine.

Amazon is the EXECUTION layer

- transparent
- optional

DATA SOURCE
────────────────────────────────────────

Primary:

Pure execution logic.

SCREEN
────────────────────────────────────────

Screen name:
CommercePreviewScreen

Route:
app/decision/commerce-preview.tsx

────────────────────────────────────────

   - Pattern icon

     "Concrete examples that match your choice"
