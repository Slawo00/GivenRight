## 🧱 TASK 1 — CREATE SUPABASE DATA ACCESS LAYER

### Create folder:

```
/services/supabase/
```

### Implement the following services:

#### 1️⃣ `uiTextService.ts`

```ts
getUIText(key: string, locale: string): Promise<string>
```

* Reads from table: `ui_texts`
* Fallback order:

  1. requested locale
  2. "en"
  3. key itself (last resort)
* Must cache results in memory per session

---

#### 2️⃣ `decisionParameterService.ts`

```ts
getDecisionParameters(): Promise<DecisionParameters>
```

* Reads from table: `decision_parameters`
* Returns a normalized object, e.g.:

```ts
{
  safe_base_weight: number,
  emotional_base_weight: number,
  bold_base_weight: number,
  occasion_multipliers: Record<string, number>,
  risk_penalties: Record<string, number>
}
```

❗ No values may be hardcoded in the engine after this step.

---

#### 3️⃣ `decisionExplanationService.ts`

```ts
getExplanation(direction: "safe" | "emotional" | "bold"): Promise<DecisionExplanation>
```

* Reads from table: `decision_explanations`
* Returns:

```ts
{
  whyThisWorks: string,
  risks: string,
  emotionalSignal: string
}
```

---

#### 4️⃣ `objectPatternService.ts`

```ts
getObjectPatterns(direction: "safe" | "emotional" | "bold"): Promise<ObjectPattern[]>
```

* Reads from table: `object_patterns`
* Returns pattern name + example object labels
* **NO marketplaces, NO brands**
